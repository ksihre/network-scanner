# Network Scanner

## Description
A basic network scanner developed using Python that identifies active hosts on a network. This project demonstrates fundamental concepts of network discovery and IP-based communication.

## Features
- Detects active devices on a network
- Performs basic host discovery
- Helps understand network communication

## Technologies Used
- Python
- Networking concepts

## How to Run
1. Make sure Python is installed  
2. Run the script  
   python scanner.py  

## Learning Outcome
- Gained understanding of network scanning techniques  
- Learned how devices communicate within a network  
- Explored basic cybersecurity concepts related to network discovery
